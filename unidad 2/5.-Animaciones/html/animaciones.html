CSS Animations

To create a transition effect, two things must be specified:

  • the CSS property you want to add an effect to
  • the duration of the effect

Transitions are controlled by using 4 properties:

  • transition-property: CSS property to be animated
  • transition-duration: in s or ms
  • transition-timing-function: bezier curve or step
  • transition-delay: time before animation starts

There is a shortcut: transition property

CSS Animations

  1. Assign a class describing animations
  2. Assign an event handler that toggles a new class describing its appearance
  3. This appearance will be achieved applying animation described at first class

When event is triggered, two classes are toggled: one for animation and other one for style

transition-timing function

Controls how animation behaves while is active. Two values: bezier curve or steps

  1. Bezier curve: animation speeds up or slows down during certain amount of time
  2. linear: constant velocity

Already provided curves:

  • ease: slow start, then fast, then end slowly (this is default)
  • linear: the same speed from start to end
  • ease-in: slow start
  • ease-out: slow end
  • ease-in-out: slow start and end
  • cubic-bezier(n,n,n,n): customized function. It can be tested at developer tools, clicking on bezier icon at CSS properties if the element has a bezier curve defined or at http://cubic-bezier.com/.

Transitionend event

when animation has ended, transitioned event is triggered. It can be used to perform an action after an animation has ended like, for instance, add another animation
