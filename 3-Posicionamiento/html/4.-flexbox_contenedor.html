Terminología básica de flex

Flex está pensado para colocar elementos en una sola línea, pero permite desbordamiento, es decir, que los elementos que no quepan se vayan a la siguiente línea.

Los elementos se organizan según la siguiente imagen

Hay que definir un contenedor con display: flex;

Aquí se muestran algunos ejemplos, no se muestran todas los valores de todas las propiedades

En todos los contenedores flex se ha usado column-gap y row-gap para dejar un espacio entre elementos sin tener que usar márgenes o rellenos

Propiedades que actúan sobre el contenedor

flex-direction y flex-wrap

Define el eje principal y controla si los elementos se pueden desbordar

Posibles valores de flex-direction: row, column, row-reverse y column-reverse

Posibles valores de flex-wrap: wrap y nowrap

flex-direction

Los elementos se organizan en una columna inversa (flex-direction: column-reverse)

justify-content

Justify-content alinea objetos en el eje principal que por defecto es el horizontal, pero se puede cambiar

Posibles valores: flex-start, flex-end, center, space-between, space-around y space-evenly

justify-content

Los elementos se organizan en una fila (flex-direction: row) y se deja un espacio equitativo entre los elementos (justify-content: space-around)

align-items

align-items sólo tiene sentido cuando le damos un tamaño al contenedor y es más grande que los elementos que contiene

Posibles valores: flex-start, flex-end, center, strech y baseline
