Propiedades de flexbox que actúan sobre los elementos

align-self

Align-self permite definir el comportamiento solo para un valor. Sólo funciona cuando los elementos ocupan todo el espacio. Valores: flex-start, flex-end, center, baseline y stretch

align-self

Los elementos se organizan en una fila (flex-direction: row) y se estiran para ocupar todo el contenedor (align-content: flex-stretch), el tercer elemento se alinea al final (align-self:end)

justify-content

Justify-content alinea objetos en el eje principal que por defecto es el horizontal, pero se puede cambiar

Posibles valores: flex-start, flex-end, center, space-between, space-around y space-evenly

justify-content

Los elementos se organizan en una fila (flex-direction: row) y se deja un espacio equitativo entre los elementos (justify-content: space-around)

align-items

align-items sólo tiene sentido cuando le damos un tamaño al contenedor y es más grande que los elementos que contiene

Posibles valores: flex-start, flex-end, center, strech y baseline

justify-content y align-items

Los elementos se organizan en una fila (flex-direction: row) se centran los elementos en su eje principal (fila) al centro (justify-content: center) y en su eje secundario (columna) arriba (align-items: start)
