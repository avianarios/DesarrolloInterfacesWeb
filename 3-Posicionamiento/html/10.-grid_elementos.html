aligning items and containers

justify-items, align-items and place-items

The first one allows to place items along the horizontal axis. values: start end center stretch

the second one allows to place items along the vertical axis. values: start end center stretch baseline. It just takes sense when vertical size of container is bigger than the addition of the size of its elements

the third one is a shortcut to place items along both axis at the same time
